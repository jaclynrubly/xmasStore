== Frequently Asked Questions ==
= How many menu does StoreXmas theme support? =
There is five Menu registered in this theme. i.e. Main Menu,  Side Menu,  Footer Menu , Top Menu and Add Social Icons Only

= Where can i get instruction about theme? = 
For detail instructions visit https://themefreesia.com/theme-instruction/storexmas/

= Is theme compatible with wpml? =
wpml-config.xml is used to support multilangual plugins such and wpml, polylang.

== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in StoreXmas in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/storexmas/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (storexmas.zip) that you have downloaded. Via FTP, upload the whole folder (storexmas) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(storexmas). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/storexmas/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the StoreXmas theme to activate it.
	- Want to enjoy StoreXmas feature. Go to  Dashboard -> Appearance -> Customize and play with StoreXmas Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.
